Fluidmesh BUNDLE (Level 3 + Level 4) Hands-On Training - New York City


Date: July 12, 2018Time: 09:00:00

Description:

Special bundle price for Level 3 and Level 4 trainings valid for two consecutive days.

Fluidmesh Level 3 training - Fixed Infrastructure
PN: FM-LEVEL3-CLASS

This training introduces architectures and implementation guidelines for a Fixed Wireless Communications Infrastructure. During this session you will learn how to choose the proper products, licenses and options for your project and how to configure and test the system for a successful deployment. More in detail:

  • Fluidmesh products and technologies dedicated to Fixed Infrastructure
  • Wireless Systems architectures (PTP, PTMP, MESH topologies)

  • User interface – wireless and networks parameters
  • RACER - the Cloud-based Radio Configuration Environment for bulk configuration of the radios
  • TITAN fast failover
  • FIPS 140-2 for Government ready deployments
  • Layer 2 support for PROFINET – one the most common protocols industrial automation
  • Command Line Interface (CLI)
  • Project rollout and installation
  • Testing and commissioning
  • Network Monitoring via SNMP
  • Hands-on training on a lab-test deployment

Fluidmesh Level 4 training - Fluidity
PN: FM-FLUID-CLASS

Fluidmesh Fluidity is the worldwide leading technology dedicated to wayside communications for moving vehicles on unlicensed bands. This training gives an in-depth overview and practical guidelines to design, implement, commission and manage a complete Fluidity system. Includes a dedicated hands-on session with variable attenuators to simulate handoff in a mobile RF environment. In detail, you will learn:

  • Fluidity and its unique value proposition - reliability, handoff and latency
  • Fluidmesh product portfolio – engineered for mobility
  • How to start a Fluidity project: site survey, RF design and approach
  • Network design - onboard the vehicle and wayside units
  • Radio configuration. Direct ETH configuration and bulk configuration through RACER
  • Project rollout - hardware installation, connections and power
  • The best tools and procedures for Testing and Commissioning
  • Hands-on - setting up and running a Fluidity lab environment with real handoff
  • Network monitoring via SNMP
  • Special applications: Yard offload with Load Balancing and Intra car communications

[Paid Event] Machine vision & deep-learning -- Intensive Developer Workshop


Date: July 16, 2018Time: 10:00:00

Description:

• What we'll do:
This is a hands-on workshop on machine vision. During this workshop, attendees​ will learn: 1) What is a convolutional neural network by building one, 2) Transfer learning, 3) Object prediction and building object prediction pipelines. We will build a neural network from scratch and train it to recognize images. Once we create a network, we will improve this model using a strategy called transfer learning. Once we understand the concepts of transfer learning, we will train a deeper network called Inception version 3 to create a state of the art image classifier. The projects will use OpenCV, Tensorflow and Keras; three very popular machine vision and deep-learning tools. We will also cover the basics of deploying scalable python applications in the cloud.

The course is hosted using either our own cloud platform: Jomiraki, a cloud connected AI developer environment or Google CoLab, a GPU powered Jupyter compatible deep-learning instance. Either of these environments will be set-up ahead of time, with zero end-user dependencies. This will ensure that each participant will spent more time testing and running the code, instead of trying to figure out the set-up process itself.

• What to bring:
This a bring your own device (BYOD) event. For optimal experience, Moad machine vision team recommends Chrome version 66 or later, to access the course contents.

• Pre-requisites:
Please set-up a Kaggle and GitHub account ahead of time. Both accounts are needed to get the full benefit of the code examples.
